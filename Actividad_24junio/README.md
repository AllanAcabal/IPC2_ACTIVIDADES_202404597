## Descripción

Este repositorio contiene la resolución de la actividad de laboratorio **"De ADO.NET Tradicional a la Automatización con Entity Framework Core"**.

La actividad aborda la diferencia entre el acceso tradicional a bases de datos mediante **ADO.NET** y el uso de **Entity Framework Core (EF Core)** como ORM para automatizar el mapeo entre objetos y tablas relacionales.

Además, se incluyen los fragmentos de código solicitados para definir un **DbContext** y realizar una consulta utilizando **LINQ**.

---

## Contenido

* Respuestas teóricas sobre la brecha de impedancia.
* Explicación de la mitigación de Inyección SQL.
* Uso de `.AsNoTracking()` para consultas de solo lectura.
* Implementación de un `DbContext`.
* Consulta LINQ utilizando Entity Framework Core.
